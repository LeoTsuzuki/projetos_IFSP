# CRUD de Clientes (Java + PostgreSQL)

## 📌 Descrição
Sistema CRUD para gerenciamento de clientes com interface gráfica.

## 🚀 Funcionalidades
- Cadastrar cliente
- Listar clientes
- Atualizar dados
- Excluir cliente

## 🛠 Tecnologias
- Java
- PostgreSQL
- JDBC

## ▶️ Como executar
1. Importar no NetBeans
2. Configurar banco de dados
3. Executar o projeto

## 🗄 Banco de Dados
Arquivo: database.sql
