## 📁 Projetos

### 🔹 CRUD de Clientes (Java + PostgreSQL)

Sistema de cadastro de clientes com operações básicas de CRUD (Create, Read, Update e Delete), desenvolvido em Java com integração ao banco de dados PostgreSQL.

**Funcionalidades:**

* Cadastrar clientes
* Listar clientes
* Atualizar informações
* Excluir clientes
